## CML Setup

- Create a CML Workspace, replace \<user-id> with your user id

  - Name: **\<user-id>**\_iceberg\_cml

  - Advanced Options

    - Instance Type: m5.4xlarge

    - GPU instances: Off

    - Enable Public IP Address for Load Balancer: checked

    - All other options can stay default
